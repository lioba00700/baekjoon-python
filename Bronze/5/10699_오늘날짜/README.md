# [Bronze 5] 오늘 날짜 - 10699

## 🔗 문제 링크

[백준 10699번: 오늘 날짜](https://www.acmicpc.net/problem/10699)

## 💡 핵심 아이디어

- **사용 모듈:** 파이썬 내장 라이브러리인 `datetime`을 활용합니다.
- **출력 형식:** 문제에서 요구하는 `YYYY-MM-DD` 형식을 맞추기 위해 `strftime("%Y-%m-%d")` 또는 f-string 포맷팅을 사용합니다.

## ⏳ 복잡도 분석

- **시간 복잡도:** $O(1)$ - 현재 시간을 한 번 조회하여 출력하기 때문입니다.

## 📝 학습 노트

- `%m`과 `%d`는 1~9월/일일 때 앞에 **0을 자동으로 채워준다**는 점을 확인했습니다.
- 파이썬의 `datetime` 모듈을 사용해 현재 날짜를 가져오는 법을 익혔습니다.
